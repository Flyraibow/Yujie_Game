Version 0.02

【SailingHero API】

## 游戏条件
+ 计算公式定义
    + 参数可以是：游戏数据、缓存数据
+ 全局命令列表回调
    + onValueChange(name, callback)

## 界面
+ 外观模板
    + 背景
    + 按钮
    + 对话框
    + 多选列表
    + 窗口
+ 命令列表回调
    + beforeShow
        + 显示前执行的命令（比如设置对话框参数）
        + 显示前 xx 游戏条件满足执行的命令（比如根据条件显示隐藏选项）
    + afterInput
        + 玩家输入后执行的命令
        + 玩家输入后 xx 游戏条件满足执行的命令

## 场景
+ 场景 = 界面包 + 音乐包

## 命令
+ 设置/显示界面元素
